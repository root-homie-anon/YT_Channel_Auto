# Description Formula — Liminal Synth

---

## Fixed Channel Hashtags

These appear on EVERY video (first 3 show above the title on YouTube):

```
#liminalsynth #synthwave #darkambient
```

---

## Description Structure

```
[One-liner: atmospheric sentence capturing the visual + sonic mood]

[Summary: 1-2 sentences about what the listener will experience]

[Chapter markers — if multi-segment]

---
[CTA line]

[Hashtags]
```

---

## One-Liner Rules

- Single sentence, under 120 characters
- Names the environment and the sonic texture — not the genre
- Reads like a scene description, not a product listing
- Examples:
  - "A rooftop above the megacity. Synth pulses dissolving into violet haze."
  - "Empty corridors beneath the overpass. Drum machines echoing off wet concrete."
  - "The skyline from below. Lush pads drifting through cold neon."

---

## Summary Rules

- 1-2 sentences max
- Mention the duration naturally ("3 hours of...", "a single unbroken hour of...")
- Reference the sonic palette without genre-labeling
- If multi-segment, mention the number of scenes
- Examples:
  - "Three hours of melancholic electronic atmospheres across 6 distinct scenes, each with its own visual world."
  - "An unbroken hour of synthesizer drifts and drum machine pulses for deep focus or late-night immersion."

---

## Chapter Marker Rules

- Only for multi-segment productions (2+ segments)
- First chapter must start at 0:00 (YouTube requirement)
- Use scene names from the title formula's Scene Name Pool
- Format: `0:00 Scene Name`
- YouTube requires minimum 3 chapters, minimum 10 seconds each

---

## CTA Line

Fixed for all videos:
```
Subscribe for more. New sessions weekly.
```

---

## Tag Construction

Generate 10-15 tags per video from these sources:

**Fixed tags (every video):**
liminal synth, synthwave, dark ambient, electronic music, ambient mix

**From music prompt:**
- Extract mood words (melancholic, atmospheric, cool, modern)
- Extract instrument references (synth, drum machine, arp, bass)
- Add tempo bucket (115 bpm, midtempo)

**From image prompt:**
- Extract environment (cyberpunk city, neon rooftop, midnight, etc.)
- Extract visual mood (cinematic, noir, futuristic)

**Duration tags:**
- Add duration bucket: "1 hour mix", "3 hour ambient", etc.

All lowercase. Max 500 characters total (YouTube limit for tags field).

---

## Topic-Specific Hashtag Construction

Generate 2-3 topic-specific hashtags beyond the fixed set:
- Pull from mood words in the music prompt
- Pull from environment in the image prompt
- Specific beats generic: `#midnightsession` beats `#music`
- All lowercase, no spaces
- Total hashtags per video: 5 max (3 fixed + 2 topic-specific)

---

## What This Formula Does NOT Do

- No shorts descriptions — music-only channels have no shorts
- No above-the-fold keyword strategy — this isn't a search-driven niche
- No SEO-first approach — the algorithm serves ambient/music channels via browse and suggested, not search
