# Description Formula (Shared)

<!-- This is the ONLY formula shared across all channels -->
<!-- All channels use this for YouTube description and hashtag generation -->

## Structure
1. First line: compelling one-liner (shows in search results)
2. Blank line
3. Video summary (2-3 sentences)
4. Blank line
5. Timestamps (auto-generated from script sections)
6. Blank line
7. Links section (subscribe, socials, related videos)
8. Blank line
9. Hashtags (3-5 relevant hashtags)

## Hashtag Rules
- First 3 hashtags appear above the title on YouTube
- Mix broad + niche hashtags
- No spaces in hashtags
- Max 15 hashtags total

## SEO Guidelines
- Include primary keyword in first sentence
- Natural language, not keyword-stuffed
- 200-500 words optimal for YouTube algorithm
